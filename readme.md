# SQL Interview Practice

This repository contains SQL interview-style problems and solutions
focused on Data Analyst and BI Developer roles.

## Topics Covered
- Joins and aggregations
- Window functions
- CTEs
- Time-based analysis
- Business-oriented SQL problems

## Tools
- SQLite
- VS Code

## Data Model
The schema simulates a simple e-commerce system with:
- customers
- orders
- order_items
- products
